Quals 2018

JST 07/28/2018 21:00 - 07/29/2018 21:00

CODE BLUE is an international security conference held in Tokyo.

In 2017, CODE BLUE started to organize its own CTF.
binja and TokyoWesterns take on the role of problem setting again this year.
But unlike last year, this time they are going to hold the Final Round
with neither Jeopardy nor Attack & Defence, but the original rule called Bulls Eye.
Don't miss it!

IRC: freenode #cbctf-2018
Twitter: @codeblue_ctf

Finals

*We will invite the following teams to CODE BLUE CTF 2018 Finals

  1. Top 9 teams in CODE BLUE CTF 2018 Quals.
  2. Top 1 team in TokyoWesterns CTF 4th 2018.

We will provide a certain amount of travel allowance for these teams.

Rule

  1. This is a team competition.
  2. There is no limit to the number of people in one team. Of course you can participate alone though.
  3. The flag format is CBCTF{something}, unless otherwise specified.
  4. You need to submit a flag to the score server in order to get a score.
  5. It is possible that one problem has multiple flags.
  6. A team that gets more score, ranks higher. If some teams get the same score, a team that made the last submission of a valid flag earlier, ranks higher among them.
  7. If you have a question, please contact us via IRC(IRC: freenode #cbctf-2018).
  8. For verification, the teams participating in the finals have to submit a writeup(your exploit code and brief description).

Prohibitions

*If you take the following actions, you will be disqualified

  1. To attack or disturb other teams
  2. To attack servers unrelated to problems
  3. To attack the infrastructure of the conference site
  4. To share flags or hints with other teams
  5. To try brute-force attacks to problem servers
  6. To use any kernel exploit in any server(especially, for solving local exploit problems)
